# 中国象棋 (Chinese Chess)

一个精美的中国象棋网页游戏，支持双人对弈和 AI 对战。

## 功能特性

- **完整的中国象棋规则**：所有棋子的移动规则、蹩马腿、堵象眼、炮翻山、将帅对面等
- **三种游戏模式**：双人对弈、电脑执红、电脑执黑
- **将军和将杀检测**：自动检测将军状态和将杀/困毙
- **悔棋功能**：支持回退走棋（AI模式下自动回退两步）
- **精美中国风界面**：仿木纹棋盘、楚河汉界、传统棋子样式
- **走棋提示**：显示可移动位置和可吃子标记
- **被吃棋子显示**：侧边栏实时显示双方被吃棋子

## 如何游玩

直接用浏览器打开 `index.html` 即可开始游戏。

## 游戏规则

| 棋子 | 走法 |
|------|------|
| 帅/将 | 在九宫内上下左右各走一步 |
| 仕/士 | 在九宫内斜走一步 |
| 相/象 | 走田字，不能过河，象眼被堵不能走 |
| 马 | 走日字，马腿被蹩不能走 |
| 车 | 横竖直线任意步 |
| 炮 | 移动同车，吃子需隔一个棋子翻山 |
| 兵/卒 | 未过河只能前进，过河后可左右移动 |

## 技术栈

- 纯 HTML5 Canvas + CSS + JavaScript
- 无任何外部依赖，单文件即可运行

## 在线体验

直接下载 `index.html` 双击打开即可。
